1. Gerak Lurus Berubah Beraturan (GLBB)

vt = v0 + a·t

1. a = Δv/Δt

2. a = (vt - v0)/t

3. a·t = vt - v0

4. vt = v0 + a·t

s = v0·t + ½a·t²

1. s = vrata-rata·t

2. vrata-rata = (v0 + vt)/2

3. vt = v0 + a·t

4. s = [(v0 + (v0 + a·t)]/2 · t

5. s = (2v0 + a·t)/2 · t

6. s = v0·t + ½a·t²

2. Gerak Parabola

tmax = v0·sinθ/g

1. Pada titik tertinggi, vy = 0

2. vy = v0y - g·t

3. 0 = v0·sinθ - g·t

4. g·t = v0·sinθ

5. t = v0·sinθ/g
